Pandas统计计算和描述

示例代码：

import numpy as np
import pandas as pd

df_obj = pd.DataFrame(np.random.randn(5,4), columns = ['a', 'b', 'c', 'd'])
print(df_obj)

运行结果：

常用的统计计算

sum, mean, max, min…

axis=0 按列统计，axis=1按行统计

skipna 排除缺失值， 默认为True

示例代码：

df_obj.sum()

df_obj.max()

df_obj.min(axis=1, skipna=False)
